# Kidney — Engineering Equivalents

This document maps each structural function to engineering substrates that can
implement the same role across synthetic, hybrid, or electrochemical systems.

## Module Equivalents
- Filtration (+1): nanoporous membranes, pressure-driven ultrafiltration
- Reabsorption (–1): microfluidic gradients, electrochemical pumps, selective membranes
- Secretion (+2): ion-injection systems, electrochemical actuators
- Countercurrent (–2): osmotic exchangers, thermal countercurrent devices
- Endocrine (+3): sensor-driven control loops, adaptive permeability systems
- Stabilization (–3): flow regulators, pressure dampers, feedback controllers
- Continuity (0): microfluidic channel design, modular connectors
- Immune (±∞): immunoisolation membranes, anti-fouling surfaces

## MVP Pathway
A minimal viable artificial kidney can be constructed by combining:
1. A +1 filtration module  
2. A –1 microfluidic reabsorption module  
3. A +2 electrochemical secretion module  
4. A –2 countercurrent exchanger  
5. A +3 adaptive control layer  
6. A –3 stabilization layer  
7. A ±∞ immune barrier
